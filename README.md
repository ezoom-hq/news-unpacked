# News Unpacked 🚀

オンラインでの議論や雑談を、ボードゲーム感覚で盛り上げるリアルタイム・Webツールです。
ZoomやDiscordなどの通話アプリと併用し、共通の「場」を作ることで、会話をより深く、楽しくします。

## ✨ 主な機能

### 1. ロビー (Welcome)
- **ワンクリック作成**: 名前を入れるだけですぐにルーム作成。
- **ワンクリック参加**: 名前とルームIDを入れるだけでルーム参加。

### 2. 準備フェーズ (Preparation)
- **お題ガチャ**: 何を話すか決まってなくても大丈夫。「基本・学習」「議論・ディベート」「政治・社会ニュース」「教養・哲学」カテゴリを選んでガチャを引けます。
- **マスク機能**: 議論が始まるまで、内容の一部分を伏せ字（？？？）で隠せます。

### 3. メインボード (Board)
- **カード選択**: 伏せられたカードの中から、ホストが議題を選んでオープン！
- **議論タイマー**: カウントダウンがあり、ダラダラしません。「+60秒」ボタンで延長も可能。
- **リアクション**: 👏 や 🎉 などの絵文字を画面上に飛ばして、言葉以外でも盛り上げられます。
- **リアルタイムチャット**: 喋りながらURLを貼ったり、メモを残したりできます。

### 4. まとめ (Summary)
- **議論ログ**: 話した内容が自動でリスト化されます。
- **コピー機能**: ワンクリックでテキストとしてコピーし、議事録として活用できます。

---

## 📖 使い方

1. **ルーム作成**: トップページから名前を入力して作成。ルームIDを参加者に共有します。
2. **お題決め**: 全員でお題を持ち寄るか、ガチャで決定します。
3. **ゲーム開始**: ホストが「ゲーム開始」を押すとボードへ移動。
4. **議論**: 気になるカードを選んでオープン！タイマーが進む中、自由に会話します。
5. **終了**: 全て終わったらサマリー画面で振り返り。
